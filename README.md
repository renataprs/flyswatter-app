# flyswatter-app
<blockquote> A simple flyswatter game developed with javascript </blockquote> </br>
</br>
<h1> Main features: </h1>
- Fly's random positions; </br>
- Fly's size random; </br>
- 3 difficulty levels based on fly's speed; </br>
- Timer and life points.</br>
